<h3 align="center">
	Ghostly Flavor for <a href="https://github.com/sxyazi/yazi">Yazi</a>
</h3>

A monochrome black & white Yazi flavor with sharp, angular design elements.

## Preview

<img src="preview.png" width="600" />

## Installation

```sh
ya pack add brandon-marella/ghostly
```

## Usage

To set it as your dark flavor, change the content of your `theme.toml` to:

```toml
[flavor]
dark = "ghostly"
light = "ghostly"
```
